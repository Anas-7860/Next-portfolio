# Anas Khan Portfolio

A clean, responsive personal portfolio built with Next.js and Tailwind CSS to showcase my projects, skills, experience, certifications, and contact links.

## Live Demo

[View the portfolio](https://anas-khan-devfolio-007.vercel.app/)

## Features

- Responsive layout for desktop and mobile
- Light and dark theme toggle
- Project, experience, education, and certification sections
- Smooth section reveals and polished UI cards
- Quick contact links for email, GitHub, LinkedIn, and LeetCode

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Deployment

This portfolio is deployed on Vercel.
